# Znft

Web mint NFT trên Solana Devnet.

## Mục tiêu
- Mint NFT trên Devnet qua web UI.
- Wallet connect + mint flow tối giản, dễ demo.

## Stack đề xuất
- Next.js + TypeScript
- Solana Wallet Adapter
- Metaplex Umi / mpl-token-metadata
